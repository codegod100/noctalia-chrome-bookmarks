# Big Clock Plugin

A simple, highly readable clock widget for the Noctalia bar.

## Features

- **Big Typeface**: Designed for readability with a bold, large font.
- **Configurable**: Change font size, time format, and show/hide seconds.
- **Theme-aware**: Automatically adjusts to your Noctalia color scheme.

## Settings

- `fontSize`: The size of the clock text (default: 24).
- `format`: The time format string (e.g., "HH:mm", "hh:mm ap").
- `showSeconds`: Whether to show seconds and update more frequently.

## Installation

This plugin is part of the standard widget collection. To enable it, ensure it's registered in your `plugins.json` and set `enabled: true`.
